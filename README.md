A simple tool to download all queries used in a Dune Analytics dashboard

## Requirements

-   [Dune API Key](https://dune.com/product/api)
-   Node.js

## Setup

Install dependencies

```bash
yarn
```

Enter your API key in a `.env` file

```bash
DUNE_API_KEY=your_dune_api_key
```

## Usage

```bash
yarn start
```
